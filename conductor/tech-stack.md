# Tech Stack

## Language & Runtime
- **Kotlin:** 2.0.10+ (JVM Toolchain 21)
- **Java:** JVM Toolchain 21 (Runtime Compatibility)

## Frameworks & Platform
- **IntelliJ Platform SDK:** IntelliJ IDEA 2022.3 - 2025.2 Support
- **Gradle IntelliJ Platform Plugin:** v2.0+

- **Gradle:** 8.12+ (Kotlin DSL)

- **Kover:** Code coverage reporting
- **Detekt:** Static code analysis for Kotlin
- **Qodana:** JetBrains quality platform
- **Gradle Changelog Plugin:** 管理された変更履歴の維持
